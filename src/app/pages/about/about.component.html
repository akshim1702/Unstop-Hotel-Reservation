Hotel Room Reservation System

Overview

Our Hotel Room Reservation System allows guests to book rooms efficiently while optimizing travel time between rooms. The system dynamically assigns rooms based on availability and minimizes guest movement within the hotel.

Hotel Structure

  • Total Rooms: 97 rooms across 10 floors.
  • Floors 1-9: Each floor has 10 rooms (e.g., Floor 1: 101-110, Floor 2: 201-210).
  • Floor 10: The top floor has 7 rooms (1001-1007).
  • Access: Each floor is accessible via a staircase and a lift located on the left side of the building.

Room Proximity & Travel Time

  • Horizontal Travel: 1 minute per room when moving between adjacent rooms on the same floor.
  • Vertical Travel: 2 minutes per floor using stairs or the lift.

Booking Rules

  1. A guest can book up to 5 rooms at a time.
  2. Priority: The system books rooms on the same floor first if available.
  3. If the required rooms aren't available on one floor, the system minimizes travel time by choosing rooms across floors.
  4. When spanning floors, the system prioritizes the lowest combined vertical and horizontal travel time.

Example Scenario

Available Rooms:

  • Floor 1: 101, 102, 105, 106
  • Floor 2: 201, 202, 203, 210
  • Floor 3: 301, 302

Case 1: If a guest wants to book 4 rooms:

The system selects 101, 102, 105, and 106 on Floor 1 because it minimizes the travel time.

Case 2: If only 2 rooms are available on Floor 1:

The system selects 101, 102 from Floor 1 and 201, 202 from Floor 2 to minimize vertical and horizontal travel time.

How to Use the System

  1. Select Room Quantity: Choose up to 5 rooms for booking.
  2. Check Availability: The system automatically checks room availability across all floors.
  3. Optimized Booking: Rooms are assigned to minimize travel time within the hotel.
  4. Confirmation: Receive a booking confirmation with room details and location.
